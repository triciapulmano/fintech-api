# Fintech API

FastAPI-based fintech API project

## Tech Stack
- FastAPI
- Python
- SQLAlchemy

## API Endpoints
**/users**
/users/register
/users/login

**/wallet**
/wallet
/wallet/add-funds

**/transactions**
/transactions/send
/transactions/history

## Features
- User registration
- User authentication
- Password hashing with bcrypt
- View wallet balance
- Money transfer
- Transactions history
